大三考试的一些回顾：
软件构造上课讲过的代码一定全部要过一遍会有挺多分  
.net的话考题会比较重复里面有一小份关于.net的复习资料很值得参考 (这门课给分好低 ToT
面向web的计算会给考点 但考的好像比较细 不要认为那种有很多很多条的就不会考...
编译原理虽然可很难 老师会给出每道题的题型的 然后也都是平时作业的题型 要认真听课做作业90+很容易拿的
软件需求工程 有点像当初考软工二 场景需求分析 画图 各种
嵌入式系统概论 只能说最后一道题特别坑 虽然老师考前给了相关信息好像最后我身边的人都觉得并没有复习到 不知道该怎么写 
我是选修课水水过了 如果是方向指选课的话 建议靠前一定要搞清楚点

J2EE与中间件 考试都是基础知识啦 不过只是选修课 没人会太认真吧

商务智能 考的还挺正常的...认真复习就好 ... 有考画多维建模模型 平时作业很重要
